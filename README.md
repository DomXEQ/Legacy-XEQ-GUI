# Legacy XEQ GUI

GUI wallet for the legacy XEQ blockchain. Pre-configured with remote nodes — no setup required.

---

## Downloads

Head to the [Releases](https://github.com/DomXEQ/Legacy-XEQ-GUI/releases) page and grab the latest version:

- **Portable** (recommended) — No installation needed. Download and run.
- **Installer** — Installs with desktop and start menu shortcuts.

---

## How to Use

1. Download the Portable `.exe` from the Releases page
2. Run it — connects to the remote node automatically
3. Wait for sync to 100%
4. Create or restore your wallet
5. Send XEQ to the swap address
6. Copy your Transaction ID
7. Go to **SWAP - TX PROOF** tab, paste your TXID and receiving address, click **Generate**
8. Copy the proof and submit on the swap portal
